# 🍅 番茄钟 - Pomodoro Timer

一个简洁优雅的番茄工作法计时器，帮助你高效管理时间，提升专注力。

## ✨ 功能

- ⏱ **工作/休息模式**：25分钟专注 / 5分钟短休息 / 15分钟长休息
- 📊 **数据统计**：记录每日番茄数和累计专注时长
- 📝 **任务列表**：管理当前待办事项
- 🔔 **完成提醒**：每个番茄时段结束自动响铃
- 🎨 **暗色主题**：护眼暗色界面，专注不被打扰
- ⌨ **键盘快捷键**：空格开始/暂停、R重置、1/2/3切换模式
- 💾 **本地存储**：统计数据自动保存到浏览器

## ⌨ 快捷键

| 按键 | 功能 |
|------|------|
| `空格` | 开始 / 暂停 |
| `R` | 重置计时器 |
| `1` | 切换到工作模式 |
| `2` | 切换到短休息模式 |
| `3` | 切换到长休息模式 |

## 🚀 使用方法

直接在浏览器中打开 `index.html` 即可使用。
